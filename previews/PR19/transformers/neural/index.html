Neural-based Encoders include categorical encoders based on neural networks:

TransformerBrief Description
EntityEmbedderEncode categorical variables into dense embedding vectors

This method has been implemented in the MLJFlux.jl which is a package that provides MLJ interfaces to deep learning models from Flux.jl. The EntityEmbedder encodes categorical features into dense vectors using a deep learning model that we must supply such as MLJFlux.NeuralNetworkClassifier or MLJFlux.NeuralNetworkRegressor.
